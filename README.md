# 🚀 RuangUsaha

### Sistem Informasi UMKM Berbasis Web

---

## 📌 Deskripsi

RuangUsaha adalah aplikasi berbasis web yang dirancang untuk membantu pelaku UMKM dalam mempromosikan usaha mereka serta memudahkan masyarakat dalam mencari dan menemukan usaha lokal.

---

## 🎯 Tujuan Pengembangan

* Digitalisasi UMKM lokal
* Mempermudah pencarian usaha
* Menyediakan platform promosi usaha

## 🛠️ Cara Menjalankan Project

1. Clone repository

```bash
git clone https://github.com/pesal21/Ruang_Usaha.git
```

2. Masuk ke folder project

```bash
cd Ruang_Usaha
```

3. Install dependency

```bash
composer install
npm install
```

4. Copy file environment

```bash
cp .env.example .env
```

5. Generate key

```bash
php artisan key:generate
```

6. Setup database di file `.env`

7. Jalankan server

```bash
php artisan serve
npm run dev
```
